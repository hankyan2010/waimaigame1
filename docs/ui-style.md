# 外卖知识答题游戏 - UI视觉规范

## 一、总原则
本项目是一个面向餐饮/外卖老板的**手机端 H5 互动页**。

视觉目标：
- 像美团官方活动页 / 官方产品页
- 干净、利落、可信、有一点游戏感
- 不花哨、不廉价、不赛博

## 二、视觉关键词
- 美团黄
- 黑 / 深灰 / 白
- 官方感
- 干净
- 直接
- 强对比
- 移动端友好
- 轻量动效

## 三、主色体系

### 主色
- 品牌黄：`#FFD100`

### 辅助色
- 标题黑：`#111111`
- 正文深灰：`#333333`
- 次级灰：`#666666`
- 边框灰：`#EAEAEA`
- 页面底色：`#F7F7F7`
- 卡片白：`#FFFFFF`
- 成功绿（少量点缀）：`#16A34A`
- 警示橙（少量提示）：`#F59E0B`
- 错误红（答错提示谨慎使用）：`#EF4444`

## 四、页面结构建议

### 1. 首页
- 顶部留白充足
- 大标题 + 副标题 + 奖励钩子
- 视觉焦点集中在开始按钮
- 可搭配店铺成长插画/图形，但不要过度复杂

### 2. 答题页
- 一屏一题
- 顶部显示题号与进度条
- 中部为题目卡片
- 下方为选项按钮
- 底部操作区保持稳定

### 3. 升级选择层
- 采用底部弹层或居中卡片弹窗
- 一次只给 3 个升级选项
- 每个选项信息要简洁：标题 + 一句话收益

### 4. 结果页
- 大分数
- 明确段位
- 结果点评要醒目
- 展示最终店铺成长状态
- CTA 按钮清晰

## 五、组件风格规范

### 按钮
- 主按钮：黄底黑字
- 次按钮：白底黑边
- 禁止渐变炫光按钮
- 圆角建议：`12px-16px`
- 高度建议：`44px-52px`

### 卡片
- 白底
- 圆角 `16px-20px`
- 轻阴影
- 边框尽量浅，避免厚重描边

### 选项
- 默认：白底深字 + 浅灰边框
- 选中：黄底 / 黄边 / 强对比
- 正确反馈：允许轻绿色点缀
- 错误反馈：允许轻红色点缀，但不要做恐吓式大红页面

### 进度条
- 底轨浅灰
- 进度部分使用美团黄
- 细而清晰，不要太厚

## 六、字体与排版

### 字体风格
- 中文系统无衬线优先
- 强调清楚、可读，不追求花字

### 字号建议
- 页面主标题：28-34px
- 二级标题：20-24px
- 正文：14-16px
- 辅助说明：12-14px
- 按钮文字：15-17px

### 排版原则
- 行距舒适
- 一屏内重点明确
- 不要堆太多字
- 避免长段说明压住交互

## 七、动效规范

### 应该有的动效
- 选项点击反馈
- 升级选择后的轻量强化反馈
- 店铺模块升级时的层级变化
- 页面切换轻微过渡

### 不应该有的动效
- 大量粒子特效
- 夸张的爆炸、发光、赛博扫描
- 连续高频闪烁
- 重型 3D 夸张转场

## 八、店铺升级主画面风格
升级体现的是**经营能力增强**，不是单纯装修越来越豪华。

建议 6 个升级模块：
1. 门头
2. 菜单
3. 后厨
4. 流量
5. 口碑
6. 会员/复购

视觉表现建议：
- 门头：招牌更清晰、更醒目
- 菜单：菜单牌更规整、更有爆款感
- 后厨：操作区更有秩序、更有效率感
- 流量：更多顾客/外卖单/流量图标出现
- 口碑：评分星级、点赞、好评标签增强
- 会员：会员卡、回头客、福利券等元素增强

## 九、禁止事项
以下风格明确禁止：
- 黑金奢华风
- 夜店霓虹风
- 赛博朋克风
- 低幼卡通风
- 传统问卷调查后台风
- 夸张营销页风格（满屏感叹号、血红字、闪瞎眼）

## 十、参考判断标准
如果一个页面看起来像：
- 美团官方活动页
- 美团商家端互动页面
- 专业但不冰冷的增长工具页

那就是对的。

如果看起来像：
- 页游广告
- 土味裂变海报
- 黑金财富课 landing page
- 传统考试系统

那就是错的。
